# MemoNotes

**MemoNotes** — это легкое и быстрое приложение для заметок для Windows, превращённое в интерактивную доску по аналогии с Miro. Размещайте текстовые блоки и изображения на бесконечном canvas, перетаскивайте их, масштабируйте доску и работайте так, как удобно вам. Приложение открывается одним кликом в углу экрана и автоматически сохраняет всю вашу работу.

## Особенности

### 🗺️ Интерактивная доска
- **Бесконечный canvas** размером 4000×4000 с точечной сеткой-фоном
- **Зум** колёсиком мыши с масштабированием к позиции курсора (10%–500%)
- **Панорамирование** средней кнопкой мыши или Ctrl + любая кнопка мыши
- **Полноэкранный режим** — кнопка ⊕ в тулбаре

### 📝 Текстовые блоки
- Добавление через кнопку 📝 в тулбаре или горячую клавишу **T**
- Двойной клик для редактирования, одиночный клик для перетаскивания
- Поддержка многострочного текста с автопереносом

### 🖼️ Изображения
- **Вставка из буфера обмена** (Ctrl+V)
- **Drag & Drop** из файлового проводника
- **Загрузка через кнопку** 🖼 в тулбаре
- Поддерживаемые форматы: PNG, JPG, JPEG, BMP, GIF, TIFF, WebP, ICO
- **Сохранение пропорций** при изменении размера (aspect ratio)

### ✋ Управление элементами
- **Перетаскивание** — свободное перемещение элементов по доске
- **Выделение рамкой** (rubber band) — выделение нескольких элементов протягиванием
- **Групповое перемещение** — перетаскивание всех выделенных элементов вместе
- **Изменение размера** — 8 ручек по углам и сторонам элемента
- **Z-порядок** — активный элемент автоматически перемещается на передний план

### 💾 Автосохранение
- Все элементы, их позиции и размеры сохраняются автоматически в `board.memo` (JSON)
- Восстанавливается масштаб, позиция на доске и все элементы между сессиями

### 🖥️ Управление окном
- **Быстрый доступ с края экрана**: наведите курсор на выбранный угол экрана, чтобы открыть доску
- **Выбор угла активации**: настройка любого из четырёх углов экрана
- **Закрепление поверх окон**: кнопка 📌 (Topmost)
- **Настраиваемая тёмная тема**: стиль VS Code
- **Запоминание размера и позиции окна**
- **Режим системного трей**: сворачивается в трей, когда не используется

## Установка

1. Клонируйте репозиторий:
    ```bash
    git clone https://github.com/max-tkv/MemoNotes.git
    ```
2. Откройте файл решения в Visual Studio, JetBrains Rider или вашем предпочитаемом .NET IDE.
3. Соберите и запустите приложение, чтобы начать.

## Использование

1. Запустите MemoNotes; оно автоматически свернётся в системный трей.
2. Наведите курсор на выбранный угол экрана, чтобы открыть доску.
3. Используйте тулбар для добавления элементов:
   - 📝 — добавить текстовый блок
   - 🖼 — вставить изображение
   - 🗑 — удалить выбранное
   - 🔄 — сбросить масштаб
4. **Колесо мыши** — масштабирование доски
5. **Перетаскивайте** элементы левой кнопкой мыши
6. **Двойной клик** по текстовому блоку — режим редактирования
7. **Выделяйте несколько элементов** протягиванием рамки на пустой области
8. Все изменения автоматически сохраняются.

### Горячие клавиши

| Клавиша | Действие |
|---------|----------|
| T | Добавить текстовый блок |
| Ctrl+V | Вставить изображение из буфера обмена |
| Delete | Удалить выбранный элемент(ы) |
| Escape | Снять выделение / выйти из редактирования |
| Колесо мыши | Масштабирование доски |
| Ctrl + ЛКМ | Панорамирование |
| Средняя кнопка мыши | Панорамирование |

### Настройки

Открыть окно настроек можно двумя способами:
- **Из окна доски**: нажмите кнопку ⚙ на панели заголовка.
- **Из системного трея**: кликните правой кнопкой мыши по иконке в трее и выберите «Настройки».

В настройках доступна:
- **Угол активации** — выбор из четырёх углов экрана, при наведении на который появляется всплывающая кнопка.
