# MemoNotes

**MemoNotes** — это легкое и быстрое приложение для заметок для Windows, превращённое в интерактивную доску по аналогии с Miro. Размещайте текстовые блоки и изображения на бесконечном canvas, перетаскивайте их, масштабируйте доску и работайте так, как удобно вам. Приложение открывается одним кликом в углу экрана и автоматически сохраняет всю вашу работу.

## Особенности

### 🗺️ Интерактивная доска
- **Бесконечный canvas** с точечной сеткой-фоном, динамически подстраивающейся под масштаб и размер окна
- **Зум** колёсиком мыши с масштабированием к позиции курсора (10%–500%)
- **Панорамирование** средней кнопкой мыши или Ctrl + любая кнопка мыши
- **Полноэкранный режим** — кнопка ⊕ в тулбаре
- **Визуальный бордер** — подсветка границ элемента при выделении
- **Ресайз окна** — возможность изменять размер окна доски вручную

### 📝 Текстовые блоки
- Добавление через кнопку 📝 в тулбаре или горячую клавишу **T**
- Двойной клик для редактирования, одиночный клик для перетаскивания
- Поддержка многострочного текста с автопереносом
- **Автоподгонка размера** — текстовое поле автоматически подстраивается под содержимое при вводе текста
- **Защита от обрезки** — при уменьшении ширины высота автоматически увеличивается, чтобы текст не скрывался

### 🖼️ Изображения
- **Вставка из буфера обмена** (Ctrl+V)
- **Drag & Drop** из файлового проводника
- **Загрузка через кнопку** 🖼 в тулбаре
- **Открытие по двойному клику** — двойной ЛКМ открывает изображение в приложении по умолчанию Windows
- Поддерживаемые форматы: PNG, JPG, JPEG, BMP, GIF, TIFF, WebP, ICO
- **Сохранение пропорций** при изменении размера (aspect ratio)

### ✏️ Рисование кистью
- **Кнопка ✏** в тулбаре или горячая клавиша **B** — включить/выключить режим рисования
- **Непрерывное рисование** — рисуйте длинные линии и фигуры без разрывов
- **Настраиваемая клавиша** непрерывного рисования в настройках
- **Свободное рисование** по пустому пространству доски левой кнопкой мыши
- **Перемещение и масштабирование** — нарисованные штрихи можно перемещать и ресайзить как изображения
- **Сохранение пропорций** при изменении размера
- **Клик по штриху** автоматически выходит из режима рисования и выделяет элемент
- **Escape** — выйти из режима рисования
- **ПКМ по штриху** — контекстное меню с удалением

### ✋ Управление элементами
- **Перетаскивание** — свободное перемещение элементов по доске
- **Выделение рамкой** (rubber band) — выделение нескольких элементов протягиванием
- **Стабильное выделение** — визуальный бордер выделения не вызывает сдвига или масштабирования изображений
- **Групповое перемещение** — перетаскивание всех выделенных элементов вместе
- **Изменение размера** — 8 ручек по углам и сторонам элемента
- **Z-порядок** — активный элемент автоматически перемещается на передний план

### ↩️ Отмена и повтор
- **Ctrl+Z** — отмена последнего действия
- **Ctrl+Shift+Z** или **Ctrl+Y** — повтор отменённого действия
- Поддержка отмены для всех операций: добавление, удаление, перемещение, изменение размера, редактирование текста
- Групповые операции (перемещение и удаление нескольких элементов) отменяются одной командой
- При фокусе в редактируемом TextBox стандартный undo/redo текста работает без перехвата

### 💾 Автосохранение
- Все элементы, их позиции и размеры сохраняются автоматически в `%APPDATA%\MemoNotes\board.memo` (JSON)
- Восстанавливается масштаб, позиция на доске и все элементы между сессиями
- При обновлении старые данные автоматически мигрируют из папки приложения в AppData

### 🖥️ Управление окном
- **Быстрый доступ с края экрана**: наведите курсор на выбранный угол экрана, чтобы открыть доску
- **Выбор угла активации**: настройка любого из четырёх углов экрана
- **Закрепление поверх окон**: кнопка 📌 (Topmost)
- **Настраиваемая тёмная тема**: стиль VS Code
- **Запоминание размера и позиции окна**
- **Режим системного трей**: сворачивается в трей, когда не используется
- **Ресайз окна** — ручное изменение размера за границы окна

### 🔄 Автоматическое обновление
- **Проверка при запуске** — приложение автоматически проверяет наличие новых релизов на GitHub
- **Уведомление** — при обнаружении обновления появляется окно с описанием изменений и кнопками действий
- **Скачивание через приложение** — прогресс-бар и автоматический запуск установщика после загрузки
- **Напомнить позже** — возможность отложить уведомление на 1 день
- **Ссылка на GitHub** — кнопка для открытия страницы релиза в браузере

## Установка

1. Клонируйте репозиторий:
    ```bash
    git clone https://github.com/max-tkv/MemoNotes.git
    ```
2. Откройте файл решения в Visual Studio, JetBrains Rider или вашем предпочитаемом .NET IDE.
3. Соберите и запустите приложение, чтобы начать.

## Использование

1. Запустите MemoNotes; оно автоматически свернётся в системный трей.
2. Наведите курсор на выбранный угол экрана, чтобы открыть доску.
3. Используйте тулбар для добавления элементов:
    - 📝 — добавить текстовый блок
    - 🖼 — вставить изображение
    - ✏ — кисть (рисование)
    - 🗑 — удалить выбранное
4. **Колесо мыши** — масштабирование доски
5. **Перетаскивайте** элементы левой кнопкой мыши
6. **Двойной клик** по текстовому блоку — режим редактирования
7. **Двойной клик** по изображению — открыть в приложении по умолчанию
8. **Выделяйте несколько элементов** протягиванием рамки на пустой области
9. Все изменения автоматически сохраняются.

### Горячие клавиши

| Клавиша | Действие |
|---------|----------|
| T | Добавить текстовый блок |
| B | Включить/выключить кисть |
| Ctrl+V | Вставить изображение из буфера обмена |
| Ctrl+Z | Отмена последнего действия |
| Ctrl+Shift+Z / Ctrl+Y | Повтор отменённого действия |
| Delete | Удалить выбранный элемент(ы) |
| Escape | Снять выделение / выйти из режима рисования / выйти из редактирования |
| Колесо мыши | Масштабирование доски |
| Ctrl + ЛКМ | Панорамирование |
| Средняя кнопка мыши | Панорамирование |

### Настройки

Открыть окно настроек можно двумя способами:
- **Из окна доски**: нажмите кнопку ⚙ на панели заголовка.
- **Из системного трея**: кликните правой кнопкой мыши по иконке в трее и выберите «Настройки».

В настройках доступна:
- **Угол активации** — выбор из четырёх углов экрана, при наведении на который появляется всплывающая кнопка.
- **Клавиша непрерывного рисования** — настройка клавиши и модификатора для непрерывного режима кисти.
