# MemoNotes

**MemoNotes** — это легкое и быстрое приложение для заметок для Windows, предназначенное для быстрого захвата мыслей, напоминаний и идей. MemoNotes открывается одним кликом в правом верхнем углу вашего экрана, мгновенно отображая минималистичное окно для ввода текста, которое позволяет вам писать и сохранять заметки без лишних отвлечений. Оно идеально подходит для пользователей, которые ищут упрощенный способ записывать быстрые заметки, оставаясь сосредоточенными на основных задачах.

## Особенности

- **Быстрый доступ с края экрана**: Наведите курсор на выбранный угол экрана, чтобы получить доступ к заметкам с плавным таймером.
- **Выбор угла активации**: Настройте любой из четырёх углов экрана (верхний левый, верхний правый, нижний левый, нижний правый) для активации всплывающей кнопки.
- **Вставка изображений**: Вставляйте изображения из буфера обмена (Ctrl+V) — они будут встроены прямо в заметку.
- **Автосохранение**: Все записи в MemoNotes автоматически сохраняются в файл `notes.memo` (формат XamlPackage), поэтому вы никогда не потеряете свои заметки, включая изображения.
- **Настраиваемая темная тема**: Наслаждайтесь стильной темной темой в стиле VS Code с кастомными элементами управления, которая будет легка для глаз при длительном использовании.
- **Минималистичный интерфейс**: Окно заметок отображает только то, что вам нужно, предлагая незагроможденное рабочее пространство.
- **Закрепление окна**: Закрепляйте окно заметок поверх всех окон с помощью кнопки 📌.
- **Быстрый доступ к настройкам**: Откройте окно настроек прямо из окна заметок, нажав кнопку ⚙ на панели заголовка.
- **Запоминание размера и позиции окна**: Приложение запоминает размер и положение своего окна, сохраняя интерфейс именно таким, каким вы его предпочитаете.
- **Режим системного трей**: MemoNotes сворачивается в системный трей, когда не используется, освобождая место на панели задач и делая его доступным, когда вам это нужно.

## Установка

1. Клонируйте репозиторий:
    ```bash
    git clone https://github.com/max-tkv/MemoNotes.git
    ```
2. Откройте файл решения в Visual Studio, JetBrains Rider или вашем предпочитаемом .NET IDE.
3. Соберите и запустите приложение, чтобы начать.

## Использование

1. Запустите MemoNotes; оно автоматически свернется в системный трей.
2. Наведите курсор на выбранный угол экрана, чтобы открыть окно для заметок.
3. Введите свою заметку. Для вставки изображения нажмите **Ctrl+V** при наличии картинки в буфере обмена — она будет встроена в текст.
4. Все изменения автоматически сохраняются в локальный файл `notes.memo` (формат XamlPackage), включая встроенные изображения.
5. Используйте кнопку 📌 на панели заголовка, чтобы закрепить окно заметок поверх всех окон.
6. Используйте кнопку ⚙ на панели заголовка, чтобы открыть настройки (выбор угла активации).
7. Когда закончите, сверните окно или дайте ему скрыться, чтобы вернуть MemoNotes в системный трей.

### Настройки

Открыть окно настроек можно двумя способами:
- **Из окна заметок**: нажмите кнопку ⚙ на панели заголовка.
- **Из системного трея**: кликните правой кнопкой мыши по иконке в трее и выберите «Открыть».

В настройках доступна:
- **Угол активации** — выбор из четырёх углов экрана, при наведении на который появляется всплывающая кнопка.
